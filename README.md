# 🐞 Miraculous — An Omarchy Theme

> *"In the daytime, I'm Marinette — just a normal girl with a normal life.
> But there's something about me that no one knows yet..."*

A dark [Omarchy](https://omarchy.dev) theme born from the collision of two worlds:
**Marinette Dupain-Cheng's** Parisian romance and **Kurt Cobain's** raw Seattle grunge.

Ladybug red meets rainy nights. Pink pastels meet fog-soaked forests.
Fashion sketches meet distorted guitars.

## Installation

### Quick (Omarchy Menu)

**Super + Alt + Space** → Install → Style → Theme → paste the repo URL.

### Manual

```bash
git clone git@github.com:ring0-rootkit/omarchy-r0-theme.git \
  ~/.config/omarchy/themes/miraculous
```

### Neovim Plugin

The `neovim.lua` points to [miraculous.nvim](https://github.com/ring0-rootkit/miraculous.nvim)
for full Treesitter + LSP editor theming. Install it separately if you want the
complete experience.
