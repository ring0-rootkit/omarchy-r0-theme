# Miraculous - Omarchy Theme

A dark theme for [Omarchy](https://omarchy.dev) inspired by **Marinette Dupain-Cheng** from *Miraculous: Tales of Ladybug & Cat Noir*.

![Preview](theme.png)

## Palette

| Color | Hex | Inspiration |
|-------|-----|-------------|
| Ladybug Red | `#E5383B` | Her suit, her power |
| Rose Pink | `#E76F8B` | Marinette's signature style |
| Blush | `#F2ACB9` | Soft and warm |
| Miraculous Gold | `#FFD166` | The Miraculous jewels |
| Seine Blue | `#7EC8E3` | The river Seine |
| Soft Lilac | `#CDB4DB` | Twilight over Montmartre |
| Stitch Green | `#95D5B2` | Thread and fabric accents |
| Ribbon Purple | `#9B5DE5` | Fashion ribbon accents |
| Parisian Night | `#0F0F1A` | Background |

## Installation

Copy the theme URL and use **Super + Alt + Space → Install → Style → Theme** in Omarchy, or manually:

```bash
git clone git@github.com:ring0-rootkit/omarchy-r0-theme.git ~/.config/omarchy/themes/miraculous
```

## Neovim

The Neovim integration uses [miraculous.nvim](https://github.com/ring0-rootkit/miraculous.nvim). Install the Neovim plugin separately for full editor theming.
